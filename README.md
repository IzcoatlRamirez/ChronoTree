
# Chrono Tree

Create your family tree quickly and easily, add photos, stories, and build your own collection of family history.


## Technologies used 

* Nodejs
* Sqlite
* React
* Typescript
* MaterialUI
* Express
* Prisma
* Jsonwebtoken
* Multer
